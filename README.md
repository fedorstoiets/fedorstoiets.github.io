## Before publishing

Search and replace:

- `YOUR-EMAIL@example.com`
- `YOUR-GITHUB-USERNAME`
- Exact license / supervision / credential language
- Real headshot
- Real testimonials only with permission

## Publish on GitHub Pages

1. Create a repo named `YOUR-GITHUB-USERNAME.github.io`.
2. Upload all files into the root.
3. Go to **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select `main` and `/root`.
6. Visit `https://YOUR-GITHUB-USERNAME.github.io`.
